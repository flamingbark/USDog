9 * 4) Static call Vat.move(user -> DaiJoin, rad)

11 * 6) Static call DaiJoin.exit(user, wad) end-to-end

13 * Usage:
14 * USER=0xYourWallet AMOUNT=0.000001 npx hardhat run scripts/debug-exit-deep.js --network bsc
